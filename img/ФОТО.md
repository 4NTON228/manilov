# Структура папок для фотографий

## img/logo/
Логотипы кафе:
- `logo-text.jpg` — текстовый логотип (используется в шапке навбара)
- `logo-oval.jpg` — овальная печать (используется в углу главного экрана)

---

## img/hero/
Фоновое фото главного экрана (hero).
Назвать файл: `hero.jpg`
Рекомендуемый размер: минимум 1920×1080px, горизонтальное фото интерьера или атмосферы кафе.

Сейчас используется внешняя ссылка — чтобы заменить, добавьте фото сюда и в index.html измените:
```
background: url('img/hero/hero.jpg') center/cover no-repeat;
```

---

## img/about/
Фото для раздела «О нас» (левая колонка).
Назвать файл: `about.jpg`
Рекомендуемый размер: 900×600px и крупнее, вертикальное или горизонтальное.

---

## img/interior/
Фотографии интерьера для раздела «Интерьер» (4 фото).
Назвать файлы:
- `interior-1.jpg` — главное фото (широкое, занимает всю высоту)
- `interior-2.jpg`
- `interior-3.jpg`
- `interior-4.jpg` — нижнее широкое фото

---

## img/news/
Фото для карточек новостей.
Назвать файлы:
- `news-1.jpg`
- `news-2.jpg`
- `news-3.jpg`

---

## img/menu/
Фотографии блюд для ленты прокрутки (внизу раздела меню) и для разделов.

### Разделы меню:
| Папка               | Раздел             |
|---------------------|--------------------|
| `img/menu/cold/`    | Холодные закуски и салаты |
| `img/menu/hot/`     | Горячие закуски    |
| `img/menu/soups/`   | Супы               |
| `img/menu/main/`    | Основные блюда     |
| `img/menu/pots/`    | В горшочках        |
| `img/menu/desserts/`| Десерты и хлеб     |
| `img/menu/drinks/`  | Напитки            |

Фото блюд для ленты кладите в соответствующую папку, называйте произвольно (например `gribochki.jpg`).
Рекомендуемый размер: 600×800px (вертикальное).
